# Billy's Adventure

Billy's Adventure is a 2D platformer built with Godot. The project focuses on classic side-scrolling gameplay: player movement, level navigation, collision handling and a simple exported game experience.

## Highlights

- 2D platformer gameplay built in Godot
- Playable character movement and level traversal
- Game assets packaged with the project
- Exported build available from the original project history
- Small game-development portfolio project

## Tech Stack

- Godot Engine
- GDScript / Godot project files
- 2D game assets

## Notes

This repository is kept as a portfolio piece for a school/personal game project. It shows practical work with a game engine, scene-based development and platformer mechanics.
